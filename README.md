### Usage

```bash
make install
make installShadowDist
```


#### Run
run with from the project directory with args

```bash
java -jar ./build/install/hospital_simulator-shadow/lib/hospital_simulator-1.0-SNAPSHOT-all.jar D,X As,I
```
or use gradle
```bash
gradle run --args "D,X As,I"
```
